# bokboot
Simple Clipboard Utility Project (For 오픈소스SW기초)

![img](image.png)


## Summary (프로젝트 안내)
Windows에서 텍스트, 파일, 디렉토리를 복사하였을때 복사한 값의 서식을 지우거나, 웹 검색을 하거나, 파일 경로를 복사하는 등의 작업을 할 수 있도록 도와주는 오픈소스 유틸리티 프로젝트
